# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python, MySQL, and Power BI. The objective is to understand customer purchasing patterns, spending behavior, and product performance through data analysis and interactive visualizations.

## Tools Used
- Python
- MySQL
- Power BI

## Files
- Customer Shopping Behavior Analysis.pdf (Project Report)
- Customer Behaviour Analysis.pbix (Power BI Dashboard)

## Key Insights
- Revenue by Gender
- Customer Segmentation
- Product Performance
- Subscription Analysis
- Shipping Type Analysis
